客户端传参解析方法文档

方法名 方法描述 备注 示例
getCity 通过回调函数的方式获取当前城市 首先通过客户端传参获取城市id；没有或不在常用城市列表中则调用http://api.map.baidu.com/location/ip的接口 
nativeAppAdapter.getCity(function(data){
	console.log(data.name) // 城市名
	console.log(data.id) // 城市名id（不一定有）
})
// retuens => the `nativeAppAdapter` object, for chain calls
getBaiduMercator 通过回调函数的方式获取当前百度墨卡托坐标 首先通过客户端传参获取坐标；没有则使用navigator.geolocation定位，再换算成百度墨卡托坐标 
nativeAppAdapter.getBaiduMercator(function(data){
	console.log('百度墨卡托坐标：'+data.x+','+data.y)
})
// retuens => the `nativeAppAdapter` object, for chain calls
getNetStatus 获取当前网络链接状态 
首先通过客户端传参获取状态；没有传参则使用navigator.onLine
全部的状态码和描述：
{
	'99':'offline',
	'100':'unknown',
	'101':'online_but_connection_type_unknown',
	'102':'2G',
	'103':'3G_or_above_3G',
	'199':'wifi'
}

nativeAppAdapter.getNetStatus()
// returns object => {
//		code:'103', // 状态码
//		describe:'3G_or_above_3G' // 网络描述
// }
getScreenCssPixels 获取当前屏幕CSS参照像素 首先通过客户端传参计算出屏幕CSS参照像素；没有则取window.screen.width|height
页面中的CSS参照像素一般不是硬件的物理像素，具体点击这里 
nativeAppAdapter.getScreenCssPixels()
// returns object => {
//		width:'320',
//		height:'480'
// }
getBduss 获取当前用户bduss 只能通过客户端传参获取；没有则为null 
nativeAppAdapter.getBduss()
// returns => string `bduss`
getMb 获取当前机型 只能通过客户端传参获取；没有则为null 
nativeAppAdapter.getMb()
// returns => string `mb`
getOs 获取当前操作系统信息 只能通过客户端传参获取；没有则为null 
nativeAppAdapter.getOs()
// returns => string `os`
getSv 获取当前地图客户端版本信息 只能通过客户端传参获取；没有则为null 
nativeAppAdapter.getSv()
// returns => string `sv`
getCuid 获取当前设备的cuid 只能通过客户端传参获取；没有则为null 
nativeAppAdapter.getCuid()
// returns => string `cuid`
